# GS BTP — Plateforme de gestion chantiers

Démo interactive d'une solution de gestion intégrée pour GS BTP (Chasseneuil-du-Poitou).

## Modules

- **Dashboard** — Vue d'ensemble des chantiers en cours, alertes, KPIs
- **Photos chantier** — Upload avec horodatage automatique, classement par chantier
- **Pointage heures** — Saisie terrain → calcul auto (jour/nuit/WE/paniers) → export intérim
- **Suivi matériel** — Parc centralisé, alertes maintenance et contrôle technique
- **Documents chantier** — Génération automatique (préparation, accueil sécurité, habilitations)
- **Rentabilité** — Suivi des coûts par chantier avec marge réelle

## Stack

HTML / CSS / JavaScript vanilla — déployable sur GitHub Pages.
